## Обо мне

Аналитик данных. Специализируюсь на сборе, очистке и визуализации данных, автоматизации отчётности и проектировании баз данных для принятия эффективных управленческих решений.

**Образование:** Бакалавр бизнес-информатики. "Московский финансово-юридический университет (МФЮА)".  
**Локация:** г. Москва



## Проекты

### **Внедрение и проектирование системы мониторинга ключевых показателей эффективности (KPI) деятельности для производственного предприятия**
**Описание:** Спроектировала БД PostgreSQL и разработала интерактивные дашборды в аналитической системе MS Power BI.

**Что сделано:**
- Проектирование структуры БД PostgreSQL
- Разработка SQL-запросов для агрегации и обработки данных
- Создание интерактивных дашбордов в Power BI

**Стек:** PostgreSQL, SQL, Power BI, BPMN 2.0, PgAdmin4, DAX 
**Направление:** BI-аналитик, продуктовый аналитик



### **Построение системы бизнес-аналитики вкладов клиентов Сбербанка**
**Описание:** Очистка и трансформация данных в Power Query, построение аналитической модели и визуализация в Power BI.

**Что сделано:**
- Сбор и очистка данных по клиентским депозитам
- Проектирование логической структуры данных
- Разработка интерактивного дашборда для мониторинга показателей
- Презентация результатов 

**Стек:** Excel, Power Query, Power BI  
**Направление:** Финансовый аналитик, Data Analyst



### **Установка и настройка программного обеспечения Astra Linux**
**Описание:** Работа в командной строке Bash, анализ клиент-серверной архитектуры и тестирование информационных систем.

**Что сделано:**
- Настройка операционной системы в виртуальной среде
- Работа с командной строкой Bash и сетевыми утилитами
- Анализ клиент-серверной архитектуры
- Написание bash-скриптов для автоматизации системной диагностики

**Стек:** Astra Linux, Bash scripting, VirtualBox  
**Направление:** Системный аналитик
