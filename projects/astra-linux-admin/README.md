# Технический анализ и администрирование Astra Linux

## Описание
Практический проект по изучению и администрированию ОС Astra Linux Special Edition.

## Что сделано
- Настройка ОС в виртуальной среде
- Работа с командной строкой Bash
- Анализ клиент-серверной архитектуры
- Тестирование информационных систем
- Написание скриптов автоматизации

## Инструменты
- Astra Linux SE
- Bash scripting
- VirtualBox
- Сетевые утилиты (ping, ssh, netstat, systemctl)
