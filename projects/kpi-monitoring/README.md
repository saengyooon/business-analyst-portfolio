# Проектирование и внедрение системы мониторинга ключевых показателей эффективности (KPI) для производственного предприятия

## Описание проекта

Проект по созданию системы мониторинга ключевых показателей эффективности (KPI) и автоматизации отчётности.

## Что сделано
- Проектирование базы данных PostgreSQL
- Написание SQL-запросов для агрегации данных
- Разработка интерактивных дашбордов в Power BI
- Настройка автоматического обновления данных
- Python-скрипт для анализа метрик

## Стек технологий
- PostgreSQL
- Power BI
- Python (Pandas, NumPy)
