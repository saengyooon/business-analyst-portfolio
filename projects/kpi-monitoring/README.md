# Проектирование и внедрение системы мониторинга ключевых показателей эффективности (KPI) для производственного предприятия

## Описание проекта

Проект по созданию системы мониторинга ключевых показателей эффективности (KPI) для компании.

## Цель

Разработка БД и внедрение АС 

## Что было сделано

- Спроектирована БД PostgreSQL 
- Написаны SQL-запросы для агрегации данных
- Разработаны интерактивные дашборды в Power BI

##  Использованные технологии

- **PostgreSQL** — проектирование БД, SQL-запросы
- **Power BI** — визуализация данных, дашборды

## Результаты

- Сокращение времени на формирование отчётов 
- Создание интерактивных дашбордов
