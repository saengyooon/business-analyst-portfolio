# Построение системы бизнес-аналитики вкладов клиентов Сбербанка

## Описание
Разработка системы бизнес-аналитики для мониторинга и анализа клиентских депозитов.

## Что сделано
- Сбор и очистка данных по клиентским депозитам
- Проектирование структуры данных
- Разработка интерактивного дашборда
- Презентация результатов представителям PIX

## Стек технологий
- Excel
- Power Query
- Power BI
