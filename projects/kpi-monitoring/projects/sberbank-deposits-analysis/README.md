# Построение системы бизнес-аналитики вкладов клиентов Сбербанка

## Описание проекта
Процесс проектирования и разработки аналитического инструментария (BI-системы) для эффективного анализа и управления вкладами.

## Цель
Разработка системы бизнес-аналитики вкладов клиентов Сбербанка на основе имеющихся данных и инструмента MS Power BI.

## Что было сделано
- Cбор и обработка данных по клиентским депозитам.
- Спроектирована структура данных и настроены процессы их подготовки.
- Разработан интерактивный дашборд для мониторинга ключевых показателей.
- Сформулированы ключевые рекомендации на основе проведенного анализа для улучшения клиентской базы и депозитной политики.

## Использованные технологии и инструменты
- **Excel** — предварительный анализ и валидация данных.
- **BI-платформа** — проектирование и визуализация данных (Power BI).
- **Документация** — описание бизнес-требований и логики расчёта метрик.

## Результаты
- Автоматизирован процесс формирования отчётности по депозитам.
- Успешная презентация работающего прототипа системы представителям PIX.
- Получен практический опыт полного цикла разработки аналитического решения: от сбора требований до визуализации.
