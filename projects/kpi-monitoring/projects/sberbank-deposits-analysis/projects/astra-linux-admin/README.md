# Технический анализ и администрирование Astra Linux

## Описание проекта
Практический проект по изучению и администрированию операционной системы Astra Linuxы. В рамках проекта проведён анализ клиент-серверной архитектуры и тестирование информационных систем.

## Цель
Получить навыки проектирования базовых и пркладных информационных систем (подсистем) и технологий.

## Что было сделано
- Установлена и настроена ОС Astra Linux на виртуальной машине.
- Освоена работа в командной строке Bash: навигация, управление файлами, права доступа, работа с процессами.
- Проведён анализ клиент-серверной архитектуры: настройка сетевых сервисов, диагностика соединений.
- Выполнено тестирование информационных систем: проверка работоспособности сервисов, анализ логов, выявление и устранение неполадок.
- Настроено взаимодействие между клиентскими и серверными компонентами системы.

## Использованные технологии и инструменты
- **Astra Linux Special Edition** — защищённая операционная система.
- **Bash** — командная строка, написание скриптов автоматизации.
- **Сетевые утилиты** — `ping`, `ssh`, `netstat`, `ip`, `systemctl` для диагностики.
- **Виртуализация** — развёртывание тестовой среды (VirtualBox / VMware).
- **Тестирование** — функциональное тестирование сервисов, анализ логов.

## Результаты
- Получены устойчивые навыки работы в командной строке Linux.
- Понимание принципов работы клиент-серверной архитектуры.
- Опыт проведения технического тестирования информационных систем.
- Готовность к работе с отечественными защищёнными ОС.
