# Игра "Крестики-нолики"

## Описание
Классическая игра крестики-нолики против компьютера.

## Как запустить
```bash
python game.py

## Правила
- Поле 3x3 с ячейками от 0 до 8
- Вы играете за крестики (X)
- Компьютер играет за нолики (O)
- Введите номер ячейки (0-8) для хода
- Побеждает тот, кто соберёт 3 в ряд

## Функции
- Простой ИИ компьютера (случайные ходы)
- Проверка победителя
- Подсчёт ничьих
- Возможность играть несколько раз
